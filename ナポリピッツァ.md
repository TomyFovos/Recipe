# コンテンポラーネアピッツァ（BIGA100%方式）レシピ

> **完成重量：約500g（2枚分）｜加水率：72%｜総粉量：約287g**  
> 工程：前日BIGA仕込み → 翌日本生地作成 → 発酵（選択式）→ 焼成

---

## このピザについて

**コンテンポラーネアピッツァ（Pizza Contemporanea）** は、2000年代にナポリで発展した現代風ナポリピザ。  
伝統的なナポリピザと異なる最大の特徴は **高く膨らんだコルニチョーネ（縁の部分）** で、外はパリッと中は雲のような軽さを持つ。  
このレシピは**全粉量をBIGAに使い切るBIGA100%方式**。本生地作成時に粉を追加せず、塩水にBIGAを溶かし込む本場イタリアのピッツェリアスタイル。

### BIGA100%方式とは

通常のBIGA法（粉の一部のみ前日仕込み）と異なり、**全粉量を前日のBIGAに使い切る**製法。  
本生地作成時は塩水にBIGAをちぎって加えて混ぜるだけ。深い風味・軽やかな食感・特徴的な大きなコルニチョーネを実現できる最本格的な方式。

**BIGA法の利点：**
- クラストがカリッとする
- 有機酸の生成による豊かな香り・旨味
- こんがりしたキツネ色になりやすい
- 大小不規則な気泡による軽い食感
- 保存性の向上

## 使用する材料の特性

| 材料 | 特徴 |
|------|------|
| **ヌーヴォラ（Nuvola）**（カプート社 0粉） | コンテンポラーネア専用粉、イタリア語で「雲」。ふわふわ軽い生地に。タンパク質12.5%、W値270-290 |
| **サッコロッソ（Sacco Rosso）**（カプート社 00粉） | 赤い袋の伝統的強力粉。グルテン補強＋小麦の旨味の要。タンパク質13.5%、W値300-320 |
| **コンテンポラーネアロッソ酵母** | 専用ドライイースト（一般ドライイーストでも代用可） |

> **配合の考え方：** ヌーヴォラ70% / サッコロッソ30%の全量をBIGAに仕込む。  
> 本生地には粉を追加しないことで、BIGAの風味を100%引き出す。

## 作り方

### 【前日】BIGA（発酵種）の準備

```
所要時間：作業5〜10分 ＋ 室温発酵4〜6時間 ＋ 冷蔵18〜24時間
```

**ステップ 1：材料混合（5〜10分）**
1. 大きめのボウルに水143.7gを入れる
2. コンテンポラーネアロッソ酵母2.9gを加えて溶かす
3. ヌーヴォラ201.2gとサッコロッソ86.2gを加える
4. 木べらで **軽く混ぜる（こねない）**
5. ⚠️ 小麦粉が水を吸い込んで **ボソボソになればOK**
6. 容器に移し、ラップで覆う（**小さな穴を数カ所開ける**）

> 💡 固めに仕込むことで風味が最大化される。BIGAは膨張するため容量に余裕のある容器を使う。

**ステップ 2：室温発酵（4〜6時間、25〜28℃）**
- 表面が泡立ち、プクプクするまで待つ
- 夏場など室温が高い時は冷蔵に移すタイミングを早める

**ステップ 3：冷蔵熟成（18〜24時間、4〜6℃）**
- 冷蔵庫でゆっくり熟成させる
- **目安：** 1.5〜1.8倍に膨らんだら完成

---

### 【翌日】本生地の作成

```
作業時間：15〜20分 ＋ 休憩20分
```

**ステップ 4：塩水を作る（2分）**
1. ボウルに水63.2gを入れる
2. 塩5.7g（＋任意で追加酵母0〜1g）を加えて溶かす

**ステップ 5：BIGA投入・混合（15〜20分）**
1. BIGAを **手で小さくちぎって** 塩水のボウルに加える
2. 手またはミキサーで混ぜる
3. **最初はドロドロで塊状になるが、15〜20分で滑らかになる**（これが"魔法"）
4. **生地温度23〜24℃** を目標にする（水温で調整）

> 💡 高加水のためベタつきやすい。手を少し濡らしながら作業するか、ミキサー使用を推奨。

**ステップ 6：生地を折りたたむ・休憩（20分）**
- 生地を数回折りたたむ（表面を張らせるように）
- ボウルに戻して **20分休ませる**

---

### 分割・発酵（2コースから選択）

**ステップ 7：分割（10分）**
- 生地を **250gずつ2個** に分割する
- 手を少し濡らしながら丸め、トレイに並べる
- **打ち粉はセモリナ粉推奨**

---

#### 🅐 短時間コース（当日焼成）

**室温で1〜2時間発酵** させたら、そのまま焼成へ。

#### 🅑 熟成コース（推奨・翌日以降）

**冷蔵庫（4〜6℃）で6〜72時間熟成**。より深い風味が生まれる。  
→ 焼成の **1〜2時間前** に冷蔵庫から出して室温に戻す。

---

### 【焼成】成形・焼成

**ステップ 8：成形（5〜10分/枚）**
1. 打ち粉（セモリナ粉）をたっぷり振った台で生地を優しく伸ばす
2. 直径 **20〜25cm**、縁を厚く残す（コルニチョーネを強調するため縁に空気を含ませる）
3. ⚠️ 高加水なので **指で押し広げるように** 伸ばす。縁の気泡を潰さない

**ステップ 9：トッピング**
- トマトソース・モッツァレラチーズ・粉チーズ・バジル・オリーブオイル

**ステップ 10：焼成（90秒〜3分、450〜500℃）**
- ピザ窯または高温オーブンを **十分に予熱**
- ピザピールで生地を移し、縁が大きく膨らみ焼き色が出るまで焼く
- 焼き上がったらすぐにカット。**熱いうちに食べるのが最適**
- 本場ナポリスタイルでは **ハサミで切り分けて** 提供

> 💡 **家庭用オーブンの場合：** 石窯シートを使い、250℃で10〜15分。

## タイムライン（全体スケジュール）

```
【短時間コース（2日）】
前日夜 20:00  BIGA作成（材料混合）
前日夜 20:10  室温発酵（4〜6時間）
前日夜 26:00  冷蔵庫へ（〜翌日18〜24時間熟成）

翌日昼 12:00  冷蔵庫からBIGA取り出し → 塩水作成 → 混合（15〜20分）
翌日昼 12:30  折りたたみ → 休憩20分
翌日昼 13:00  分割・丸め
翌日昼 13:00  室温発酵（1〜2時間）
翌日昼 14:30  成形 → 焼成

──────────────────────

【熟成コース（3日以上・推奨）】
前日夜 20:00  BIGA作成・室温発酵 → 冷蔵
翌日昼 12:00  本生地作成 → 分割 → 冷蔵庫へ（6〜72時間熟成）
焼成当日      冷蔵庫から取り出し（1〜2時間前）→ 復温 → 成形 → 焼成
```

## ポイント・注意事項

| ポイント | 内容 |
|---------|------|
| BIGAはボソボソでOK | まとまりにくい固めの状態が正解。こねると発酵が乱れる |
| BIGAは膨張する | 容量に余裕のある容器を使う。ラップに小穴を開けてガス抜き |
| 生地温度23〜24℃を目標 | 本生地作成時、水温で調整して生地が熱くなりすぎないように |
| 混合は15〜20分が目安 | 最初はドロドロだが必ず滑らかになる。焦らず混ぜ続ける |
| 成形時は気泡を潰さない | コルニチョーネの気泡は絶対に潰さない。指で押し広げる |
| 焼成温度は450〜500℃ | 家庭用は石窯シート使用・250℃で10〜15分が目安 |
| ベタつく場合 | セモリナ粉を多めに使用、または冷蔵熟成を延長する |
| 夏場・高温時 | 発酵が進みすぎるためBIGAを早めに冷蔵へ移す |

## トラブルシューティング

| 症状 | 対処法 |
|------|--------|
| 膨らまない | イーストを増やす（最大2g追加）。室温発酵時間を延ばす |
| 発酵しすぎた | 冷蔵庫で発酵を止め、早めに使用する |
| ベタつきが強い | セモリナ粉を多めに使用。冷蔵熟成を延長 |
| コルニチョーネが膨らまない | 成形時に縁の気泡を確認。焼成温度を上げる |
| 生地が破れる | 混合時間を15〜20分しっかり取ってグルテン形成を完了させる |

---

## 保存方法

- **冷蔵：** 分割後の生地玉を密閉容器で **最大72時間**
- **冷凍：** 分割後の生地玉を個別にラップして **最大1ヶ月**  
  → 使用 **12時間前** に冷蔵庫に移して解凍し、焼成1〜2時間前に室温に戻す
