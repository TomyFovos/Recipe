# コンテンポラーネアピッツァ（BIGA法・風味重視）レシピ

> **完成重量：約500g（2〜3枚分）｜加水率：72%｜総粉量：300g**  
> 工程：前日BIGA仕込み → 翌日本生地作成 → 分割後1〜2日冷蔵 → 焼成

---

## このピザについて

**コンテンポラーネアピッツァ（Pizza Contemporanea）** は、2000年代にナポリで発展した現代風ナポリピザ。  
伝統的なナポリピザと異なる最大の特徴は **高く膨らんだコルニチョーネ（縁の部分）** で、外はパリッと中は雲のような軽さを持つ。  
このレシピは**サッコロッソをBIGAに多く配合することで、小麦のナッティな旨味を強調した風味重視版**。

### BIGA法とは

小麦粉・水・酵母を低温でゆっくり発酵させる伝統的なイタリアの発酵種（前種）。  
通常の生地状にせず **固めでまとまりにくい状態** で仕込むことで温度上昇を防ぎ、有機酸が生まれ豊かな香りと旨味を生む。

**BIGA法の利点：**
- クラストがカリッとする
- 有機酸の生成による豊かな香り・旨味
- こんがりしたキツネ色になりやすい
- 大小不規則な気泡による軽い食感
- 保存性の向上

## 使用する材料の特性

| 材料 | 特徴 |
|------|------|
| **ヌーヴォラ（Nuvola）**（カプート社 0粉） | コンテンポラーネア専用粉、イタリア語で「雲」。ふわふわ軽い生地に。タンパク質12.5%、W値270-290 |
| **サッコロッソ（Sacco Rosso）**（カプート社 00粉） | 赤い袋の伝統的強力粉。グルテン補強＋小麦の旨味の要。タンパク質13.5%、W値300-320。**Tipo1で代用可**（全粒粉風の深みがさらに増す） |
| **ドライイースト** | 一般ドライイーストで可。総量は粉量の0.33%と少量で風味を出す |

> **配合の考え方：** BIGAにサッコロッソ40%を使い旨味の骨格を作り、本生地添加粉をヌーヴォラ100%にすることで軽さを確保。  
> ヌーヴォラ全体76.7%・サッコロッソ23.3%のバランスで風味と軽さを両立。

## 作り方

### 【前日】BIGA（発酵種）の準備

```
所要時間：作業10分 ＋ 室温発酵2〜3時間 ＋ 冷蔵18〜48時間
```

**ステップ 1：材料混合（10分）**
1. 大きなボウルにヌーヴォラ105gとサッコロッソ70gを入れ、ドライイースト0.75gを全体に振りかける
2. 水95gを少しずつ加えながら、フォークやヘラで混ぜる（粉が全体に湿る程度まで）
3. ⚠️ **こねない（重要）**。生地は固めで、まとまりにくい状態が理想
4. ラップをかける

> 💡 **Biga伝統の作り方：** 固めに仕込むことで風味が最大化される。粉の比率を厳守する。

**ステップ 2：室温発酵（2〜3時間、25〜27℃）**
- 生地が1.5倍程度に膨らむまで待つ
- 湿度が高い日はイーストを0.5gに減らす

**ステップ 3：冷蔵熟成（18〜24時間、4〜6℃。最長48時間可）**
- 密閉容器に移し冷蔵庫でゆっくり熟成させる
- **目安：** 表面が気泡だらけになり、酸味の効いた小麦の香りが出ればOK
- 翌日以降の使用が風味的に最適

---

### 【翌日】本生地の作成

```
作業時間：20〜30分 ＋ 室温発酵30〜60分
```

**ステップ 4：水の1段階目投入（5〜10分）**
1. BIGA全量（約270.75g）をボウルに入れ、追加ドライイースト0.25gを振りかける
2. 水120gの **半量（60g）** を加えて軽く混ぜ、生地をほぐす

**ステップ 5：塩水の投入（5分）**
- 塩7.5gを残りの水60gに溶かし、生地に **徐々に加える**
- 全体でここまで5〜10分かけてゆっくりなじませる

**ステップ 6：粉合わせ（5〜10分）**
- ヌーヴォラ125gを **3回に分けて** 加える
- 台の上に移して手ごね（またはミキサー低速5分 → 高速3分）
- ⚠️ **ウィンドウテスト合格まで** こねる（生地を薄く伸ばして光が透ければOK）
- ベタつく場合は水を10g減らす。**捏ねすぎ厳禁（風味が飛ぶ）**

> 💡 高加水のためベタつきやすい。手をオリーブオイルでコーティング、またはミキサー使用を推奨。

**ステップ 7：一次発酵（30〜60分、室温）**
- 生地を丸めてボウルに戻し、オリーブオイル少量を表面に塗ってラップをする
- 生地が **1.2倍** に膨らむまで待つ（膨らみすぎ注意）

---

### 分割・二次冷蔵熟成

```
作業時間：10分 ＋ 冷蔵12〜24時間
```

**ステップ 8：デガス・ベンチタイム（10分）**
- 生地を軽くデガス（手でやさしく押して空気を抜く）
- そのまま**10分ベンチタイム**（休ませる）

**ステップ 9：分割・冷蔵二次発酵（12〜24時間）**
- 生地を2〜3等分（各 **160〜250g**）に分割し、丸める
- **打ち粉はセモリナ粉推奨**（くっつき防止）
- 密閉容器に入れ、冷蔵庫（4〜6℃）で12〜24時間二次発酵させる
- 長時間ほど小麦の旨味が増す

> 💡 500gを1枚で使う場合は分割せずそのまま発酵させてもOK。

---

### 【焼成当日】成形・焼成

**ステップ 10：復温（1時間前、室温）**
- 冷蔵庫から出し **1時間** かけて室温に戻す

**ステップ 11：成形（10分/枚）**
1. 打ち粉（セモリナ粉）をたっぷり振った台で生地を優しく伸ばす
2. 直径 **20〜25cm**、縁を厚く残す（コルニチョーネを強調するため縁に空気を含ませる）
3. ⚠️ 高加水なので **指で押し広げるように** 伸ばす。縁の気泡を潰さない

**ステップ 12：トッピング**
- トマトソース・モッツァレラチーズ・粉チーズ・バジル・オリーブオイル

**ステップ 13：焼成（90秒〜2分、450℃以上）**
- ピザ窯または高温オーブンを **十分に予熱**
- ピザピールで生地を移し、縁が大きく膨らみ焼き色が出るまで焼く
- 焼き上がったらすぐにカット。**熱いうちに食べるのが最適**
- 本場ナポリスタイルでは **ハサミで切り分けて** 提供

> 💡 **家庭用オーブンの場合：** 石窯シートを使い、250℃で10〜15分。  
> コルニチョーネの高さが **15〜20mm** になれば成功の目安。

## ポイント・注意事項

| ポイント | 内容 |
|---------|------|
| BIGAは「固めに・こねない」 | まとまりにくい固めの状態が正解。こねると発酵が乱れる |
| 水は2段階で加える | 本生地作成時、先に半量→塩水を残り半量で溶かして添加 |
| 粉は3回に分けて投入 | 一度に入れるとダマになりやすい |
| ウィンドウテストまでこねる | 薄く伸ばして光が透ければグルテン形成完了。捏ねすぎ注意 |
| 成形時は気泡を潰さない | コルニチョーネの大きな気泡は絶対に潰さない。指で押し広げる |
| 焼成温度は450℃以上 | 家庭用は石窯シート使用で250℃・10〜15分が目安 |
| 水の調整 | 湿度が高い日は水を5g減らす。ベタつく場合もヌーヴォラを少量追加 |
| 室温が高い場合 | イーストをさらに減らして過発酵を防ぐ |

---

## トラブルシューティング

| 症状 | 対処法 |
|------|--------|
| 膨らまない | イーストを少し増やす。室温発酵時間を延ばす |
| 固くなる | 発酵時間を延ばす（冷蔵をさらに半日追加） |
| ベタつきが強い | 水を10g減らす。手をオリーブオイルでコーティング |
| コルニチョーネが膨らまない | 成形時に縁に空気を十分含ませる。気泡を潰していないか確認 |

---

## 保存方法

- 未使用の分割生地は **冷凍保存可能（2週間以内）**
- 解凍後、室温で **1時間復温** してから成形・焼成
