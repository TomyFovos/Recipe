# コンテンポラーネアピッツァ（BIGA法）レシピ

> **完成重量：500g（約2枚分）｜加水率：72%**  
> 2日工程：前日にBIGA仕込み → 当日に本生地作成・焼成

---

## このピザについて

**コンテンポラーネアピッツァ（Pizza Contemporanea）** は、2000年代にナポリで発展した現代風ナポリピザ。  
伝統的なナポリピザと異なる最大の特徴は **高く膨らんだコルニチョーネ（縁の部分）** で、外はパリッと中は雲のような軽さを持つ。

### BIGA法とは

小麦粉・水・酵母を **48時間以上かけて低温発酵** させる伝統的なイタリアの発酵種（前種）。  
通常の生地状にせず **ちぎった状態** で発酵させることで温度上昇を防ぎ、理想的な熟成を実現する。

**BIGA法の利点：**
- クラストがカリッとする
- 有機酸の生成による豊かな香り
- こんがりしたキツネ色になりやすい
- 大小不規則な気泡による軽い食感
- 保存性の向上

## 使用する材料の特性

| 材料 | 特徴 |
|------|------|
| **ヌーヴォラ（Nuvola）**（カプート社 0粉） | コンテンポラーネア専用粉、イタリア語で「雲」。ふわふわ軽い生地に。タンパク質12.5%、W値270-290 |
| **サッコロッソ（Sacco Rosso）**（カプート社 00粉） | 赤い袋の伝統的強力粉。グルテン構造の補強役。タンパク質13.5%、W値300-320 |
| **コンテンポラーネアロッソ酵母** | 専用ドライイースト（製品仕様に応じて調整。一般ドライイーストでも代用可） |

> **配合理由：** ヌーヴォラ70%で軽さを出しつつ、サッコロッソ30%で強度を補う。  
> 高加水率72%でも扱いやすく、理想的なコルニチョーネが形成される。

## 作り方

### 【前日】BIGA（発酵種）の準備

```
所要時間：約20〜26時間（作業5分＋発酵2〜4時間＋冷蔵18〜24時間）
```

**ステップ 1：材料混合（5分）**
1. 室温の水にコンテンポラーネアロッソ酵母を溶かす
2. ヌーヴォラとサッコロッソを加えて **軽く混ぜる**
3. ⚠️ **こねない（重要）**。粉が全体になじめばOK
4. ちぎった状態（ゴロゴロした塊）のままにしておく

**ステップ 2：室温発酵（2〜4時間、25〜28℃）**
- 表面が泡立ち、少し膨らんでくるまで待つ

**ステップ 3：冷蔵熟成（18〜24時間、4〜6℃）**
- 冷蔵庫に入れてゆっくり熟成させる
- 風味と酸味が深まる

---

### 【当日】本生地の作成

```
作業時間：約45〜60分（＋発酵・熟成時間）
```

**ステップ 4：BIGA分割（5分）**
- 冷蔵庫から出したBIGAを **手で小さくちぎる**
- 扱いやすいサイズ（2〜3cm程度）にする

**ステップ 5：水合わせ（2分）**
- BIGAに追加の水を加えてなじませる

**ステップ 6：粉合わせ（3分）**
- 追加のヌーヴォラとサッコロッソを加えて混ぜる

**ステップ 7：塩・酵母追加（2分）**
- 塩と追加の酵母を投入してよく混ぜ込む

**ステップ 8：こね（10〜15分）**
- 手ごねまたはミキサーで **滑らかになるまで** こねる
- 生地が艶やかでまとまったらOK

**ステップ 9：室温発酵（1.5〜2時間、25〜28℃）**
- 1.5〜2倍の大きさに膨らむまで発酵させる

**ステップ 10：冷蔵熟成（24〜72時間、4〜6℃）** ← ここが風味の決め手
- 長時間ほど風味・香りが向上する
- 最短24時間、理想は48〜72時間

---

### 【焼成当日】仕上げ・焼成

**ステップ 11：復温（30〜60分、室温）**
- 冷蔵庫から出して室温に戻す

**ステップ 12：分割・成形（10分）**
- 約250gずつに分割する（2枚分）
- ⚠️ **コルニチョーネの気泡を潰さないよう** 優しく丸める
- 高加水生地のため、生地を潰さないよう注意

**ステップ 13：最終発酵（30〜45分、室温）**
- ふっくらするまで発酵させる

**ステップ 14：焼成（90秒〜3分、450〜500℃）**
- ピザ窯または高温オーブンで焼成
- 🌡️ **温度が低い場合（家庭用オーブン）は時間を延ばして調整**
- 本場ナポリではハサミで切り分けて提供する

## タイムライン（全体スケジュール）

```
前日夜 20:00  BIGA作成（材料混合）
前日夜 22:00  冷蔵庫へ（〜翌日夜まで熟成）

翌日昼 12:00  冷蔵庫からBIGA取り出し → 本生地作成
翌日昼 14:00  本生地の室温発酵完了
翌日昼 14:00  再び冷蔵庫へ（24〜72時間熟成）

──（24〜72時間後）──

焼成当日      冷蔵庫から取り出し復温 → 成形 → 最終発酵 → 焼成
```

---

## ポイント・注意事項

| ポイント | 内容 |
|---------|------|
| BIGAは「こねない」 | こねすぎると発酵が乱れる。軽く混ぜるだけでOK |
| 高加水のため扱いが難しい | 成形時は優しく。コルニチョーネの大きな気泡を絶対に潰さない |
| 焼成温度は高めが必須 | 450℃以上が理想。家庭用オーブンの場合は予熱を十分に |
| 冷蔵熟成は長めに | 最低24時間、できれば48〜72時間でより豊かな風味になる |
| 酵母の調整 | コンテンポラーネアロッソ酵母がない場合は一般ドライイーストで代用可。製品仕様に応じて量を調整 |
